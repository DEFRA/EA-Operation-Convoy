Category - Conversion Format
-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-

 - The format to which the input file that is getting converted to.
Accepts only drop down values, any manual edit to the property will
throw a runtime error.

Please visit Office support to check compatibility

-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-
Category - Input File path
-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-

- Protected/Encrypted files will not be converted  - the files
that are not corrupted and accessible to the bot will be converted.

Formats accepted - *.doc;*.docx;*.rtf;

CAUTION : Conversion Requires the Word Application to be installed in
the system/VM.
